Icon Generator

To create icons for the extension:

  1. Create a 128x128 PNG icon with the TaskNotes logo/clipboard emoji
  2. Use an online tool like icon-generator.net
  3. Generate 16x16, 32x32, 48x48, and 128x128 versions
  4. Name them: icon-16.png, icon-32.png, icon-48.png, icon-128.png
  5. Place them in this icons/ folder

For now, you can use simple colored squares or the browser will show a default icon.
